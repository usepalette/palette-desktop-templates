# Presentation Drafts

Drafts are markdown files that outline a presentation. They're the thinking/planning stage before generating HTML slides.

## Template

```markdown
# Presentation Title

**Event:** Event name
**Date:** YYYY-MM-DD
**Duration:** X minutes
**Audience:** Who are they, what do they care about

## Key message

One sentence: what should the audience walk away with?

## Outline

### 1. Opening (X min)
- Hook / opening question
- Why this matters to the audience

### 2. Section title (X min)
- Key point
- Supporting detail / example
- Speaker notes: _what to say, not what's on the slide_

### 3. Section title (X min)
- Key point
- Supporting detail / example

### N. Close (X min)
- Recap key message
- Call to action / next step
- Q&A prompt

## Slide notes

Per-slide content ideas — what goes ON the slide (visuals, text, data) vs what you SAY.

| # | Slide content | Speaker notes |
|---|---|---|
| 1 | Title + subtitle | Introduce yourself briefly |
| 2 | ... | ... |
```
