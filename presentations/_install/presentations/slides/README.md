# Slides

HTML presentations. Open in a browser and press **F** to go full-screen.

## Controls

- **→** / **Space** / **click right half** — next slide
- **←** / **click left half** — previous slide
- **N** — open the presenter window (speaker notes, next-slide preview, timer)
- **T** — toggle dark / light theme
- **F** — toggle fullscreen
- **Escape** — exit fullscreen
